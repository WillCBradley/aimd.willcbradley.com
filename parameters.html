Parameters

Experience is a valuable teacher.

Like doctors learning from each case, ML models use each piece of training data to refine their predictions.

One way of doing this is through parameters.

Parameters are internal, modifiable variables within a model, used to produce predictions based on inputs. The two most common types are:

  • Weights — these tweak the relative importance of different input features in the overall prediction
  • Biases — these enable the model to alter its predictions independent of weights

When models us parameters, we call it "parametric machine learning" (such as in linear or logistic regression).

But other algorithms, like the decision trees we discussed earlier, are "non-parametric". These models don't assume anything about the form of the training data, and therefore don't use parameters to specify anything about their functions.

In both cases, models can still have "hyper-parameters".

These have set values before any data is seen, alter how the model interacts with data. Examples include maximum depth of a decision tree, or learning rate in gradient descent.
